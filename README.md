Este proyecto consiste en el desarrollo del backend de una aplicación de e-commerce, construido con Spring Boot y diseñado bajo una arquitectura en capas que permite mantener el código organizado, escalable y fácil de mantener.

El objetivo principal es simular el funcionamiento de una tienda online desde el lado del servidor, implementando una API RESTful capaz de gestionar usuarios, productos y operaciones básicas del sistema. A través de distintos endpoints, se permite realizar acciones como el registro e inicio de sesión de usuarios, la autenticación mediante tokens JWT y la administración de recursos dentro de la plataforma.

Uno de los focos del proyecto es la implementación de seguridad utilizando Spring Security, integrando autenticación basada en JSON Web Tokens para proteger los endpoints y garantizar que solo usuarios autorizados puedan acceder a determinadas funcionalidades.

Además, se hace uso de Spring Data JPA para la persistencia de datos en una base de datos relacional, facilitando la interacción con la misma mediante repositorios y evitando la complejidad de consultas manuales.

En conjunto, el proyecto busca aplicar buenas prácticas del desarrollo backend moderno, combinando tecnologías del ecosistema Java con conceptos fundamentales como APIs REST, separación de responsabilidades, manejo de datos y seguridad, sentando una base sólida para futuras ampliaciones como carrito de compras, pagos online o gestión avanzada de usuarios.
